Home Page

 I am Zarna Bharodia, pursing Masters in Geographic Information Science and Technology at North Carolina State University, North Carolina. Part of my GIS 550 Geospatial Data Structures and Web Services coursework I developed web portfolio.

This Geospatial Data Structures and Web Services (GIS 550) course focuses on spatial and tabular data structures, implementation and dissemination issues. Content includes comprehensive review of database conceptual, logical and physical design; using web services to deliver map and tabular data; open source and commercial software tools; and current Geographic Information Science issues.

Part of this coursework the ePorfolio is developed and delivers through this website. The ePortfolio covers following topics. Each main topic is elaborate through different sections in this website.

  • Geodatabase Design
  • Commercial Web Delivery
  • Open Source GIS
  • Comprehensive Project Development (Capstone Portfolio)
